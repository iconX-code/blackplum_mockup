# blackplum mockup — Dependencies

## Runtime (CDN)
React 18              https://unpkg.com/react@18/umd/react.production.min.js
ReactDOM 18           https://unpkg.com/react-dom@18/umd/react-dom.production.min.js
Babel Standalone      https://unpkg.com/@babel/standalone/babel.min.js
Hanken Grotesk        https://fonts.googleapis.com/css2?family=Hanken+Grotesk:wght@400;500;600;700;800;900&display=swap
                      (display 폰트 — 헤드라인/버튼/eyebrow/nav)
Inter                 https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap
                      (body 폰트 — 본문/lead/running text)
Pretendard Variable   https://cdn.jsdelivr.net/gh/orioncactus/pretendard/dist/web/variable/pretendardvariable.css
                      (한글 fallback — JSX 카피가 한국어이므로 Hanken/Inter의 한글 글리프 부재 보완. spec.md §3-7-1)

## 자체 자산 (정적 파일)
styles.css            3-tier 디자인 토큰 + 컴포넌트 스타일
mock-data.js          window.MOCK_DATA — 메시지/스레드/게시글/매크로 룰
icon-set.js           window.ICON_SET — 아이콘 name → SVG path mapping
                      1차 출처: Lucide (MIT 라이선스). 필요한 SVG path만 발췌하므로 외부 라이브러리 런타임 의존 없음

## Dev Dependencies (npm — 개발 도구 전용, 배포에 포함되지 않음)

@eslint/js            eslint v10에서 별도 분리된 권장 룰 셋 (flat config의 js.configs.recommended)
eslint-plugin-html    HTML 내 embedded script 린트 지원
globals               languageOptions.globals 표준 정의 (browser globals 등)
